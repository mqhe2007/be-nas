# be-nas

[![skills.sh](https://skills.sh/b/mqhe2007/be-nas)](https://skills.sh/b/mqhe2007/be-nas)

## 简介
be-nas 让你化身一位精通本机自建 NAS 的专家。当你需要处理家庭 NAS 的搭建、初始化、配置或运维问题时，可使用此技能并基于 Docker 部署服务套件。典型场景包括硬件选型与裸机直装、存储池与 SMB/NFS 文件共享、影视库与媒体自动化、家庭相册、下载管理，以及权限隔离、备份与同步策略。

## 安装方法

```bash
npx skills add mqhe2007/be-nas
```

