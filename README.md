## 简介
be-nas 让任意支持 skills 的 AI Agent 化身精通本机自建 NAS 的专家。当你需要处理家庭 NAS 的搭建、初始化、配置或运维问题时，be-nas 能提供专业、系统、可执行的方案，帮助你把家用主机、迷你主机、旧电脑或服务器配置为稳定、安全、可维护的家庭 NAS。无论是硬件选型、系统安装、存储规划，还是共享配置、媒体库管理、下载工具设置，be-nas 都能给出详细步骤，并提供安全、严谨的实施与验证方法，确保你的家庭 NAS 运行顺畅。

## 安全原则

- 任何格式化磁盘、修改分区、重建阵列、删除数据、开放公网访问或提升系统权限的操作，都应先盘点环境、说明风险并获得明确确认。
- 默认优先保护数据、备份和回滚路径；不把 RAID、LVM 或同步误当作备份。
- 示例使用通用占位符，避免在公开配置、文档或问题描述中暴露真实用户名、序列号、密钥、公网 IP 等个人信息。
